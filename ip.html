關於 IP 的小知識

網際網路協定（英語：Internet Protocol，縮寫：IP）是是網際網路協定套件中的網路層通訊協定，用於跨網路邊界封包交換。它的路由功能實現了互聯互通，並從本質上建立了網際網路。

💻 常見的名詞

  • 固定 IP
  • 浮動 IP
  • 實體 IP (對外/外網)
  • 虛擬 IP (對內/內網)

用生活的例子解釋

實體 IP 就像送子鳥的電話總機，虛擬 IP 就像是分機號碼。

外部的人可以打給總機，再請總機轉接到分機；內部的人可以直接打分機號碼。

💻 查看 IP 的方法

搜尋 myip 點選任一網站即可查看目前的「實體 IP」

📌 Windows

開啟命令提示字元，輸入 ipconfig。

📌 macOS

開啟終端機，輸入 ifconfig。
