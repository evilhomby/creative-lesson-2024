關於 CSS 的 Position

CSS 的 Position 屬性是用來控制元素的位置，預設值為 static。其他有 4 個值可以選擇：

  • relative: 相對位置
  • absolute: 絕對位置
  • fixed: 固定位置
  • sticky: 粘性位置
.position-relative {
    position: relative;
}

.position-absolute {
    position: absolute;
}

.position-fixed {
    position: fixed;
}

.position-sticky {
    position: sticky;
}

📌 預設位置與設定位置

設定除了 static (靜態) 以外的值都可以透過 top, right, bottom, left 來調整位置。若未設定預設仍會在「原本的位置」上。

📌 絕對定位元素脫離文件流與填充元素寬高

設定 absolute, fixed 時即為「絕對定位元素」，其特性會脫離文件流，不再佔據原有的空間

沒有設定 width, height 時會依照內容決定元素的尺寸。但可以透過設定 top+bottom 與 left+right 來填滿元素的寬高。

<style>
div {
    background: #f00;
    position: absolute;
    /* 同時設定 top bottom 時可以填充元素的高度 */
    top: 0px;
    bottom: 0px;
    /* 同時設定 left right 時可以填充元素的寬度 */
    left: 0px;
    right: 0px;
}
</style>
<!-- 一個佔滿可視範圍的紅色區塊-->
<div></div>

📌 參考基準

  • relative: 相對於元素原本的位置
  • absolute: 相對於最近的「已定位」祖先元素，即設定不是 static 的元素
  • fixed: 相對於瀏覽器視窗 (參考基準不會因為捲動捲軸而改變)
  • sticky: 相對於滾動容器的內容

📌 深度 (z-index)

當有多個元素重疊時，可以透過 z-index 來控制元素的深度。數字越大，元素越在上層。 沒有設定 z-index 時 (預設值為 auto)，愈後出現的元素在愈上方 (但它們仍在同一層)。

深度有群組的概念。

/* 出現在文字下一層 */
.nagtive {
    position: absolute;
    z-index: -1;
}

📌 使用 Excel 產生隨機色碼

使用 RANDBETWEEN 函數產生隨機數字，以下範例產生 0 到 255 之間的數字。

=RANDBETWEEN(0,255)

應用情境：產生 rgba(r, g, b, a)

=CONCATENATE("rgba(",RANDBETWEEN(0, 255),",",RANDBETWEEN(0, 255),",",RANDBETWEEN(0, 255),",",RANDBETWEEN(0, 100),")")

應用情境：產生 hex 色碼

=CONCATENATE("#",DEC2HEX(RANDBETWEEN(0,255),2),DEC2HEX(RANDBETWEEN(0,255),2),DEC2HEX(RANDBETWEEN(0,255),2))

應用情境：產生隨機位置

="top:"&RANDBETWEEN(0,800)&"px;left:"&RANDBETWEEN(0,1920)&"px;"

重點提示

CONCATENATE 函數可以將多個字串合併，或是使用 & 符號來連接字串。
