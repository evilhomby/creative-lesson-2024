前端開發利器 VSCode

工欲善其事，必先利其器。VSCode 是一款由微軟開發的開源程式碼編輯器，支援 Windows、macOS。它具有強大的擴充功能，可以滿足前端開發的各種需求，也是目前前端開發者最喜愛的編輯器之一。

💻 安裝 VSCode

至官網下載📦對應系統的安裝檔進行安裝，目前版本為 1.86 版。

實作練習

在個人電腦上完成 VSCode 1.86 的安裝

💻 VSCode Portable Mode (可攜式)

可攜式用於無法安裝 VSCode 的電腦，或是需要同時開啟多個 VSCode 的情況，例如比對安裝了某個擴充套件或設定之間的差異。

Win

於官網下載 zip 檔案，解壓縮後於資料夾內新增 data 資料夾，即完成啟用 Portable Mode。

Mac

下載 zip 檔案後解壓縮，於 Visual Studio Code.app 同層新增一個資料夾並命名為 code-portable-data。執行如果仍不是 Portable Mode 則需於終端機執行以下命令

xattr -dr com.apple.quarantine Visual\ Studio\ Code.app

💡 基本設定建議

遵循「少即是多」的原則，將軟體介面上不必要的元素隱藏，專注於程式碼編輯。

VSCode 的設定有兩種方式，一種是使用 UI 介面進行設定，另一種是直接修改 settings.json 檔案。

開啟設定 UI 的兩種方式：

  1. 使用快捷鍵 ctrl/cmd + , 開啟設定 UI。
  2. 使用快捷鍵 ctrl/cmd + shift + p 輸入 settings 於列表中選擇 Preferences: Open Settings (UI)。

開啟 settings.json 的兩種方式：

  1. 開啟設定 UI 後，點選右上角的文件的圖示。
  2. 使用快捷鍵 ctrl/cmd + shift + p 輸入 settings 於列表中選擇 Preferences: Open Settings (JSON)。

以下是一些基本的設定建議，可以直接複製貼上至 settings.json 套用設定。

{
    "breadcrumbs.enabled": false,
    "editor.bracketPairColorization.enabled": false,
    "editor.fontSize": 28,
    "editor.inlineSuggest.enabled": true,
    "editor.multiCursorModifier": "ctrlCmd",
    "editor.renderLineHighlight": "gutter",
    "editor.smoothScrolling": true,
    "explorer.autoReveal": false,
    "explorer.compactFolders": false,
    "explorer.confirmDelete": false,
    "search.mode": "newEditor",
    "window.commandCenter": false,
    "window.newWindowDimensions": "inherit",
    "workbench.colorTheme": "Monokai",
    "workbench.editor.empty.hint": "hidden",
    "workbench.editor.showIcons": false,
    "workbench.editor.untitled.labelFormat": "name",
    "workbench.settings.applyToAllProfiles": [
        "window.title"
    ],
    "workbench.startupEditor": "none",
    "workbench.tips.enabled": false,
    "workbench.tree.indent": 40,
    "workbench.tree.renderIndentGuides": "none",
}

💻 快速鍵

VSCode 有許多快速鍵可以提高開發效率，以下是 VSCode 官方提供的常用快速鍵對照表。

如果是從 Sublime Text 轉換過來的開發者，可以安裝 Sublime Text Keymap and Settings Importer 擴充套件，可以讓你在 VSCode 中使用 Sublime Text 的快速鍵。

必備的快速鍵為 ctrl/cmd + shift + p 開啟命令列，可以快速搜尋並執行命令。

💻 自訂 Snippets

Snippets 是一種程式碼片段，可以幫助你快速輸入重複的程式碼。VSCode 內建了許多 Snippets，也可以自訂 Snippets。

命令列輸入 snippets 選擇 Preferences: Configure User Snippets > New Global Snippets File，輸入檔案名稱後即可開始編輯 Snippets。

$ 開頭為變數或是佔位符。

{
	"工作日誌": {
		"prefix": "log",
		"body": [
			"# $CURRENT_YEAR-$CURRENT_MONTH-$CURRENT_DATE",
			"",
			"## 工作內容",
			"",
			"$1",
			"",
			"## 挑戰與學習",
			"",
			"$2"
		],
		"description": "這是自訂的工作日誌片段"
	}
}

實作

建立一個簡易的工作日誌片段，當你輸入 log 時，自動展開成一個工作日誌的格式， 包含日期、時間、工作內容 (挑戰與學習)…等。

重點提示

良好的原始碼縮排可以提高可讀性，VSCode 內建 Reindent Lines 命令，可以快速調整程式碼的縮排。
