Emmet 是什麼？

Emmet 是前端開發的工具包，主要的功能包含：

  • 利用 CSS 選擇器來快速產生 HTML 的開發語法
  • 透過縮寫產生樣式
  • HTML, CSS 的快速變更行為：例如：更改標籤，包裏元素…等

VSCode 內建 Emmet 不需額外安裝，輸入片段或縮寫後按下 Tab 鍵即可展開； 行為操作可以於選單中找到或是使用快速鍵。

📌 產生 HTML 片段

在 html 檔案中輸入以下縮寫 (abbreviation)後按下 TAB 鍵展開

.container>header+main+footer

可以產生以下 HTML：

<div class="container">
    <header></header>
    <main></main>
    <footer></footer>
</div>

小技巧

如果按 TAB 鍵沒有反應可以改用以下幾種方式：

  • 按 ctrl+space 後出現 Emmet 提示時按 TAB 鍵展開
  • 刪除幾個字重新輸入後再按 TAB 鍵展開
  • 使用選單列「Edit > Emmet: Expand Abbreviation」
  • 按 ctrl/cmd+shift+p 開啟命令列，輸入 Emmet: Expand Abbreviation 來展開

實作

請用 emmet 產生以下 HTML 片段

<html>
    <body>
        <header></header>
        <main>
            <div class="container">
                <div class="row">
                    <div class="col-2">01</div>
                    <div class="col-2">02</div>
                    <div class="col-2">03</div>
                    <div class="col-2">04</div>
                    <div class="col-2">05</div>
                    <div class="col-2">06</div>
                </div>
            </div>
        </main>
        <footer></footer>
    </body>
</html>
點我看答案
html>body>header+(main>.container>.row>.col-2{$$}*6)+footer

html>body>header+main>.container>.row>.col-2{$$}*6^^^footer

TIP

使用 Emmet 產生 HTML 時不需要一次產生完整結構，可以拆分多個部分逐步產生。重點是透過縮寫展開省去輸入的時間。

📌 假字產生器

Emmet 提供 lorem 產生假字的功能 (可惜沒有中文)，快速產生填充用的文字，若於最後接上數字則代表產生指定的單詞數量。

lorem5 > Lorem ipsum dolor sit amet.

📌 產生樣式

在 css 檔案或是 <style></style> 內輸入縮寫後按下 TAB 鍵展開， 例如：

m0 > margin: 0;
p0 > padding: 0;
w100 > width: 100px;
h100p > height: 100%;

CheatSheet

可以參考 Emmet CheatSheet 來了解更多的縮寫。

📌 HTML, CSS 的快速變更行為

按 ctrl/cmd+shift+p 開啟命令列，輸入 Emmet 即可看到所有相關的命令，較常使用的有：

  • Wrap with Abbreviation : 將選取的元素包裏在指定的標籤內
  • Update Tag : 更改標籤名稱

可以個人使用習慣設定快速鍵來提高效率。

重點提示

可以安裝 VSCode 的 Auto Rename Tag 來更改標籤名稱，效率比 Emmet: Update Tag 更高。
